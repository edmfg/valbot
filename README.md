# Orbit · v0.1

Audience constellation prototype for Google's B2B Small Business targeting. Visualises the relationship between a target audience, the LinkedIn topics they engage with, and the Reddit communities they inhabit outside of work.

## Run

Open `index.html` in any modern browser. No build step. No backend.

## Deploy

Push to `main` and enable GitHub Pages on the repo (Settings → Pages → Source: `main` / root).

## Stack

- D3.js v7 (force simulation, drag, SVG)
- Vanilla JS, single file
- Inter via Google Fonts
